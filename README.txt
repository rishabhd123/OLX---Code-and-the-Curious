Instructions to generate ads_recommendation.csv file

1. Put all the training and test files in the folder Data with the same name as given in the competition.
2. Execute Solution1.ipynb using jupyter notebook(this will generate subm.csv).
3. Execute Solution_Metapath.ipynb using jupyter notebook(this will generate sub_mat.csv).
4. Execute Ensemble.ipynb using jupyter notebook(this will generate ads_recommendation.csv).



Language: python-2.

IDE: jupyter notebook.

Libraries Used
pandas, numpy, sklearn, scipy
